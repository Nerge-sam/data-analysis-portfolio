# Titanic Survival Data Analysis

This project is an exploratory data analysis of the Titanic passenger dataset. The goal was to identify the key factors that influenced a passenger's chance of survival.

## Key Findings
* **Sex:** Being female was the strongest predictor of survival (~73% survival rate).
* **Passenger Class (Pclass):** 1st class passengers (~62%) were far more likely to survive than 3rd class passengers (~26%).
* **Age:** Children (under 18) had a higher survival rate (~53%) than adults (~36%).
